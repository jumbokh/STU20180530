# scikit-learn 使用範例

上面的 Linear Regression, SVM, 跟 Kmeans 的例子，主要是取自[政大蔡炎龍老師的講義](http://bit.ly/py3data2018)。

上面的例子主要是用來強調scikit-learn的機器學習模型的使用方式：

1. 載入模型
2. 建立模型
3. 訓練模型 (fit)
4. 使用模型 (predict)

我們先講解了在資料完整的狀況下的機器學習模型的使用，接著我們說明如何對使用者資料做前處理。

資料前處理的使用方式概略如下：

1. 載入類別
2. 建立物件
3. 計算參數 (fit)
4. 轉換資料 (Transform)
